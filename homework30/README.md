# Weather Telegram Bot 🌤️

Це Python-проєкт, який використовує зовнішній API OpenWeather для отримання погодних даних та Telegram Bot API для взаємодії з користувачем.

---

## 🔧 Використані технології

- Python
- requests
- python-telegram-bot
- python-dotenv

---

## 🌐 Використані API

- OpenWeather API — отримання поточної погоди та прогнозу
- Telegram Bot API — робота бота

---

## 🚀 Можливості бота

Бот підтримує:

### 📍 Прогноз на сьогодні
- Поточна температура
- Відчувається як
- Мін/макс температура
- Стан погоди
- Вітер
- Хмарність
- Поточна дата

### 📅 Прогноз на 5 днів
- Прогноз по днях
- Температура
- Відчувається як
- Стан погоди
- Вітер
- Хмарність

---

## 🤖 Команди

- /start — запуск бота
- /help — довідка
- /weather — прогноз на сьогодні
- /forecast — прогноз на 5 днів

Також доступні кнопки:
- "Прогноз на сьогодні"
- "Прогноз на 5 днів"

---

## ▶️ Як запустити проект

### 1. Клонувати репозиторій

git clone <your-repo-url>
cd <project-folder>

---

### 2. Створити .env файл

OPENWEATHER_API_KEY=ваш_ключ  
TELEGRAM_BOT_TOKEN=ваш_токен

---

### 3. Встановити залежності

pip install -r requirements.txt

---

### 4. Запустити бота

python bot.py

---

## 💻 Консольна версія

Також є окрема програма:

python weather_console.py

Вона дозволяє отримувати погоду через консоль.
